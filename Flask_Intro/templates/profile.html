Welcome to your profile section of this Flask-powered utility website. This site provides a collection of simple yet functional web tools designed to perform basic text and numeric operations such as converting strings to uppercase and computing geometric areas such as Circles and Triangles.

The purpose of this platform is to demonstrate clean integration between Python back-end logic and a responsive Bootstrap front-end, showing how data can move dynamically between user input, Flask routes, and displayed results.

Each page in this project represents a separate feature, helping users explore how modular web applications are structured while keeping usability simple and direct.
